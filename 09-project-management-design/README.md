# 09 - Navrh systemu spravy projektu pro male tymy

Projekt obsahuje UML navrh a staticke HTML wireframy.

## UML

- `uml/use-case.puml` - diagram pripadu uziti.
- `uml/activity-task-flow.puml` - diagram aktivit pro praci s ukolem.
- `uml/state-task.puml` - stavovy diagram ukolu.

## Wireframy

Otevrit `wireframes/index.html` v prohlizeci. Obsahuje:

- domovskou stranku,
- stranku projektu,
- rozhrani pro pridani ukolu,
- nastenku pro sledovani pokroku.
