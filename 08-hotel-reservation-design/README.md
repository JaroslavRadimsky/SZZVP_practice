# 08 - Navrh rezervacniho systemu pro male hotely a penziony

Projekt obsahuje UML navrh a staticke HTML wireframy.

## UML

- `uml/use-case.puml` - diagram pripadu uziti.
- `uml/sequence-reservation.puml` - sekvencni diagram rezervace pokoje.
- `uml/class-diagram.puml` - tridni diagram domeny.

## Wireframy

Otevrit `wireframes/index.html` v prohlizeci. Obsahuje:

- hlavni stranku,
- stranku rezervace,
- profil uzivatele,
- formular pro zpetnou vazbu.
